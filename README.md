WasteDetectorWEB is a **web-based intelligent waste classification system** built with **TensorFlow/Keras** for training and **Flask** for deployment.  
The project allows users to upload an image of waste, and the system will predict its category in real time.

ğŸ“Š Model Training
The CNN is trained with the TrashNet dataset (trashnet/data/dataset-resized).

- Image size: 128x128 pixels.

- Batch size: 32.

- Epochs: 100.

- Data augmentation: rotation, shifts, flips.

The trained model is stored as:

```bash
modelo_basura.h5
```
During testing, the model reached an accuracy of ~85% depending on the dataset split.

## ğŸŒ Sharing the Web App with ngrok

By default, the Flask application runs locally at: http://127.0.0.1:5000/


This means it is only accessible from your own computer.  
If you want to **share your app over the internet** (for testing, demos, or remote access), you can use **[ngrok](https://ngrok.com/)**.  

### â–¶ï¸ Steps

1. **Download and install ngrok**  
   - Go to [https://ngrok.com/download](https://ngrok.com/download)  
   - Extract the downloaded file and add it to your system PATH.  

2. **Run your Flask app**  
   ```bash
   python app.py
   ```
By default, Flask will be running on port 5000.

Expose the Flask port with ngrok
In another terminal, run:

```bash
ngrok http 5000
```
Copy the public URL generated by ngrok
You will see something like:

```nginx
Forwarding    https://abc123.ngrok.io -> http://127.0.0.1:5000
```

Use the https://abc123.ngrok.io link to access your Flask app from anywhere.

âš ï¸ Notes
- The free version of ngrok generates a temporary URL each time you start it.

- For a persistent domain, you need to create a free/paid ngrok account and configure an auth token.

- Do not expose sensitive apps without proper security, as ngrok makes your local app publicly accessible.
